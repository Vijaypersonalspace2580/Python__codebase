# 💰 Personal Finance & Budget Management System

A console-based Personal Finance and Budget Management System developed using Python.

This application allows users to create accounts, set monthly budgets, track income, record expenses, and monitor remaining balance using CSV-based persistent storage.

---

## 🚀 Features

- 🔐 User Account Creation & Login System
- 📊 Set Monthly Budget
- ➕ Add Additional Income
- ➖ Add Expenses with Categories
- 📂 Store Data Using CSV Files
- 📈 View Expense Summary (Category-wise)
- 💵 View Remaining Budget Balance
- 🛡 Input Validation & Error Handling
- 📁 Automatic File Creation

---

## 🛠 Technologies Used

- Python
- CSV Module
- File Handling
- Dictionaries
- Conditional Logic
- Exception Handling
- CLI (Command Line Interface)
